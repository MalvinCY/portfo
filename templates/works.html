001 / 006

Academic Paper Recommender

Python SPECTER2 FAISS FastAPI Streamlit NLP

A content-based academic paper recommendation system built on SPECTER2 transformer embeddings and FAISS approximate nearest-neighbour search. The system indexes 9,280 research papers across 6 ML/AI categories (Computer Vision, NLP, Reinforcement Learning, Graph Neural Networks, Recommendation Systems, and Transformers).

Users can input a paper title, abstract, or free-text query and receive the most semantically similar papers ranked by cosine similarity. The project includes a live Streamlit web app for interactive exploration and a FastAPI REST API for programmatic access.

002 / 006

Plant Health Detector

Python PyTorch MobileNetV2 Transfer Learning Streamlit Computer Vision

A deep learning image classification model that identifies 38 different plant disease conditions across 14 crop species. Built using MobileNetV2 with transfer learning, the model achieves 93.96% accuracy on the test set.

The training pipeline includes data augmentation, learning rate scheduling, and early stopping. A live Streamlit web app allows users to upload leaf images and receive instant disease predictions with confidence scores and treatment recommendations.

GitHub Live Demo
003 / 006

Patient Sentiment Classifier

Python PyTorch LSTM Word2Vec FastAPI Docker NLP

An NLP sentiment analysis system trained on 215,000 patient drug reviews from the UCI ML Repository. The model uses a 2-layer bidirectional LSTM with pre-trained Word2Vec embeddings to classify reviews into positive, neutral, or negative sentiment.

The pipeline includes text preprocessing (tokenisation, stopword removal, lemmatisation), custom Word2Vec training, and hyperparameter tuning. Deployed as a Dockerised FastAPI REST API achieving 73.5% accuracy across 3 sentiment classes.

004 / 006

Weather ETL Pipeline

Python Apache Airflow AWS S3 EC2 ETL REST API

An end-to-end ETL (Extract, Transform, Load) data pipeline that extracts real-time weather data from the OpenWeatherMap API, transforms it into a structured format with derived metrics, and loads the results into an AWS S3 bucket for storage and analysis.

The pipeline is orchestrated using Apache Airflow running on an AWS EC2 instance, with scheduled DAGs handling extraction, transformation, and loading as separate, monitored tasks. Includes error handling, logging, and data validation steps.

005 / 006

Covid Global Impact Analysis

SQL Tableau Data Analysis Visualisation

A comprehensive SQL-driven analysis of global Covid-19 data, exploring mortality rates, infection spread, and vaccination progress across countries and continents. The analysis uses advanced SQL techniques including CTEs, window functions, temp tables, and aggregate functions.

Key insights are presented through an interactive Tableau dashboard that allows users to filter by country, date range, and metric type. The dashboard provides at-a-glance views of infection rates, death percentages, and vaccination rollout timelines.

TFL Bus Safety Dashboard

Tableau Data Visualisation Transport Data Analytics

An interactive Tableau dashboard visualising Transport for London (TFL) bus safety incident data across all London boroughs. The dashboard enables users to explore accident reports by borough, time period, incident type, and severity level.

Features include geographic heatmaps, trend analysis over time, and drill-down capabilities for identifying high-risk routes and areas. Built to support data-driven decision-making for urban transport safety improvements.
