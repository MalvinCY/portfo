# Portfolio Website

Personal portfolio website for Malvin Siew — Clinical Data Analyst.

**Live:** [mcysiew.eu.pythonanywhere.com](https://mcysiew.eu.pythonanywhere.com/)

## About

A Flask-based portfolio showcasing projects in data analytics, machine learning, and clinical data engineering. Built on the Mashup Template (Bootstrap 3) with custom styling.

## Features

- **Projects** — Six project cards with modal popups, interactive Tableau embeds, and links to GitHub repos and live demos
- **Publications** — Peer-reviewed research in Nature Communications, Cellular Microbiology, and BMJ Innovations
- **Contact form** — Submissions saved to CSV with email notifications via Yahoo SMTP

## Tech Stack

- Python / Flask
- HTML, CSS, JavaScript
- Bootstrap 3 (Mashup Template)
- Tableau Public (embedded dashboards)
- PythonAnywhere (hosting)

## Local Development

```bash
pip install flask
python server.py
```
